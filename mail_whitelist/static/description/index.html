Mail whitelist

This module enables you to set up a list of emails that can receive notifications from the system.

  • Overview
  • Releases
  • When only specific emails or domains should be allowed to receive letters from the system. Others won't be able to receive, reducing the potential for mistakes.
  • When you need to test the notification submissions and letters can only be submitted to the configured email address.

Verify Odoo website for Google services

Description
Google services require verifying site ownership for a website. This can be done in several ways, in Odoo the simplest - by using this module. Just fill in your HTML meta tag code in Odoo and click on "Verify domain" on the Google side. It can take a few minutes only.

Configuration
Actions on Google side
During the "Verify and claim website" procedure choose the option "Add an HTML tag". Then copy a value from the "content" parameter to the clipboard.
Actions in Odoo
Activate the Developer Mode and go to the menu "Website" - "Configuration" - "Websites" to fill in the "Verification Code" field.
Then go back to the Google settings and click on "Verify website".
